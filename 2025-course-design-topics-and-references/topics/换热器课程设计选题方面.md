# 换热器课程设计创新研究

**主题：基于峰谷电价的电蓄热供暖换热系统优化设计**

---

## 1. 研究背景与意义

我国电力系统存在明显的峰谷电差价。充分利用电网低谷时段电能，通过**电蓄热技术**储能，在高峰时段释放，可有效实现**削峰填谷、提高能源利用率、降低供暖成本**。

该思路不仅符合节能减排政策要求，也为“电-热”联合供能提供了技术支撑。

---

## 2. 分时电价特性分析

根据广东地区工商业电价标准（如图所示），可分为：

* **平段、低谷、高峰时段**；
* **大工业用电**：包括“基本电价”和“电度电价”；
* **农业用电**：单一电价，政策扶持。

利用此特性，在低谷时段运行加热系统储热，在高峰时段关闭电源、释放热量，从而节省电费。

---

## 3. 蓄热方式对比分析

| 蓄热方式 | 蓄热材料     | 优点         | 缺点         | 投资费用 |
| ---- | -------- | ---------- | ---------- | ---- |
| 水蓄热  | 热水       | 成熟可靠，系统简单  | 占地大，温度受限   | 低    |
| 固体蓄热 | 氧化镁、铁、砖等 | 温度高，可直接供蒸汽 | 成本高，寿命有限   | 高    |
| 相变蓄热 | 石蜡、盐类    | 热密度高，温度波动小 | 导热率低、相分离风险 | 中    |

> 可见，相变蓄热在性能与经济性上折中，是当前研究热点。

---

## 4. 系统设计与工作原理

### 4.1 电制热蓄热系统

**工作流程**（如图1）：

1. 低温空气 → 加热丝（电能转热） → 蓄热体储能；
2. 热空气进入换热器 → 加热水介质；
3. 热水从出水口流出，完成循环供热。

系统实现了“电 - 热 - 储 - 热 - 用”的闭环，适用于**工业热水、供暖系统**。

---

### 4.2 混凝土蓄热模块（CTES）

铜管包裹混凝土，翅片结构增强换热效率（见图2）。
混凝土蓄热容量大，可实现**热能的大规模储存与释放**，适合**峰谷电储热系统**。

---

### 4.3 谷电相变供暖系统

组成部分：

* **电阻丝**：谷电时段通电发热；
* **相变蓄热单元**：储存热能；
* **变频风机与气水换热器**：实现热量释放供暖；
* **控制单元**：实现定时储热与放热。

此系统实现“低谷储热 + 高峰放热”的智能控制，**兼顾经济性与舒适性**。

---

### 4.4 相变热库系统

热库外观为立式设备，内部填充相变材料，外围有保温层。
通过水循环完成热量传递，实现**稳定输出与高效储能**。

---

## 5. 空气-水管翅式换热器设计

| 参数    | 要求                      |
| ----- | ----------------------- |
| 热负荷范围 | 20–60 kW                |
| 热风温度  | 28–90 °C                |
| 风机流量  | 8500–25000 m³/h         |
| 全压    | 300–400 Pa              |
| 换热器尺寸 | 1300×1300 mm            |
| 循环水泵  | 5 t/h，扬程 12 m，功率 0.5 kW |

换热器结构紧凑、换热面积大，适合**中小型集中供暖系统**。

---

## 6. 设计任务与创新要点

设计内容包括：

1. 确定地区峰谷电差价；
2. 明确应用场合（建筑、小区、厂房等）；
3. 选择换热器形式；
4. 计算换热面积、换热系数；
5. 进行经济性分析与总结。
